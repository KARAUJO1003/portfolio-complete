# Web App

Next.js app para portfolio publico e admin.

Estrutura principal:

```txt
src/
├─ app/
├─ core/
├─ features/
├─ components/
├─ providers/
├─ hooks/
└─ themes/
```

Regras:

- Use `features/` para dominios de produto.
- Use `core/` para infraestrutura.
- Nao customize `components/ui` para produto.
- Use `components/ds` para wrappers e componentes proprios.
